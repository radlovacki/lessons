# Прелазне секвенце

| Секвенца | Назив           | UNICODE                        |
|----------|-----------------|--------------------------------|
| `\'`     | Single quote    | 0x0027                         |
| `\"`     | Double quote    | 0x0022                         |
| `\\`     | Backslash       | 0x005C                         |
| `\0`     | Null            | 0x0000                         |
| `\a`     | Alert           | 0x0007                         |
| `\b`     | Backspace       | 0x0008                         |
| `\e`     | Escape          | 0x001B                         |
| `\f`     | Form feed       | 0x000C                         |
| `\n`     | New line        | 0x000A                         |
| `\r`     | Carriage return | 0x000D                         |
| `\t`     | Horizontal tab  | 0x0009                         |
| `\v`     | Vertical tab    | 0x000B                         |
| `\u`     | UTF-16          | `\uHHHH` (0000 - FFFF)         |
| `\U`     | UTF-32          | `\U00HHHHHH` (000000 - 10FFFF) |
| `\x`     | UTF-x           | `\xH[H][H][H]` (0 - FFFF)      |

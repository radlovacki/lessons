Елемент div

Увод

Елемент div је генерички контејнер за HTML елементе. Нема утицаја на садржај или распоред елемената у њему, све док се на неки начин не стилизује помоћу CSS-а.

Атрибути id и class

Као контејнер, елемент div не представља ништа - он се користи за груписање садржаја у комбинацији са атрибутима class или id. id се користи као јединствен идентификатор (само једном на страници), док се class може користити са више елемената.

Више о div елементу можеш прочитати на MDN-у.
